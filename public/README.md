# Public Assets

This folder contains static assets for your portfolio.

## Profile Image

Please add your profile picture here as `profile.jpg` or `profile.png`.

**Recommended specifications:**
- Format: JPG or PNG
- Size: 600x600px or larger (square aspect ratio)
- File size: Under 500KB for optimal performance
- Quality: High resolution for crisp display

**File name:** `profile.jpg` or `profile.png`

The image will be automatically optimized by Next.js Image component.
